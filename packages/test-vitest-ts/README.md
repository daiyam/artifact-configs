The configuration to do unit testing with [vitest](https://vitest.dev/) and [typescript](https://www.typescriptlang.org/).

Getting Started
---------------

```sh
mkdir myproject
cd myproject
npm init -y
artifact add @daiyam/lang-js @daiyam/lang-ts @daiyam/npm @daiyam/npm-ts @daiyam/test-vitest-ts
```
