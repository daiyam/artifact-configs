The configuration to do unit testing with [mocha](https://mochajs.org/) and [typescript](https://www.typescriptlang.org/).

Getting Started
---------------

```sh
mkdir myproject
cd myproject
npm init -y
artifact add @daiyam/base @daiyam/lang-ts @daiyam/npm-ts @daiyam/test-mocha-ts
```
