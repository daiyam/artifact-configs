The configuration to have me as the author of the created node module.

Getting Started
---------------

```sh
mkdir myproject
cd myproject
npm init -y
artifact add @daiyam/base @daiyam/lang-ts @daiyam/npm-ts @daiyam/npm-daiyam
```
