The configuration to do unit testing for VSCode extensions.

Getting Started
---------------

```sh
mkdir myproject
cd myproject
npm init -y
artifact add @daiyam/lang-js @daiyam/lang-ts @daiyam/vsx-ts @daiyam/vsx-test-mocha
```
