[![NPM Version](https://img.shields.io/npm/v/@daiyam/artifact-gh-issuehub.svg?colorB=green)](https://www.npmjs.com/package/@daiyam/artifact-gh-issuehub)

The configuration to use GitHub Issues as an issue tracker and accept issue report in GitHub Discussions.

Getting Started
---------------

```sh
mkdir myproject
cd myproject
npm init -y
artifact add @daiyam/gh-issuehub
```
